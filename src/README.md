
# Dashboard Tutor Virtual PI2

Archivo React listo para integrar en tu repositorio.

## Incluye
- `DashboardTutorVirtualPI2.jsx`: componente principal del dashboard
- `README_subida_github.md`: instrucciones rápidas de integración

## Requisitos
- React
- Tailwind CSS
- `framer-motion`
- `lucide-react`

## Instalación rápida

```bash
npm install framer-motion lucide-react
```

## Uso

Importa el componente donde quieras mostrarlo:

```jsx
import DashboardTutorVirtualPI2 from "./DashboardTutorVirtualPI2";

export default function App() {
  return <DashboardTutorVirtualPI2 />;
}
```
