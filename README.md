# Recurso Tutor Virtual 1

Sitio estatico listo para publicar en GitHub Pages.

## Contenido

- `index.html`: estructura principal del recurso
- `styles.css`: estilos responsivos del sitio
- `script.js`: contenido e interactividad
- `src/`: entrega original basada en React incluida como referencia

## Publicacion

El sitio puede desplegarse directamente desde la raiz del repositorio en GitHub Pages.
